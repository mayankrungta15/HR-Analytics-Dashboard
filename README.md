# HR Analytics Dashboard

Interactive HR Analytics Dashboard built using Power BI.

## Objective
Analyze employee attrition and workforce metrics to identify key factors affecting employee turnover.

## Key KPIs
- Total Employees
- Active Employees
- Attrition Rate
- Average Salary
- Average Age
- Employees Left

## Features
- Department-wise Attrition Analysis
- Job Role Analysis
- Age Group and Gender Analysis
- Education Field Analysis
- Business Travel Impact
- Job Satisfaction Analysis
- Overtime Analysis
- Key Influencers (AI Visual)
- Smart Narrative

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query
